inigo.intranetpolicy Installation
---------------------------------

To install inigo.intranetpolicy using zc.buildout and the plone.recipe.zope2instance
recipe to manage your project, you can do this:

* Add ``inigo.intranetpolicy`` to the list of eggs to install, e.g.:

    [buildout]
    ...
    eggs =
        ...
        inigo.intranetpolicy

* Re-run buildout, e.g. with:

    $ ./bin/buildout
